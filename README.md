# 🎵 Spotify Song Popularity Predictor

## 📌 Overview
This project predicts the popularity of a Spotify song using machine learning models like XGBoost and LightGBM.

It solves:
- Regression → Predict popularity score (0–100)
- Classification → Predict whether a song is a hit

---

## 🚀 Features
- End-to-end ML pipeline
- Feature engineering
- Model comparison (XGBoost vs LightGBM)
- SHAP explainability
- Data visualization

---

## 🛠 Tech Stack
- Python
- Scikit-learn
- XGBoost
- LightGBM
- SHAP
- Matplotlib / Seaborn

---

## 📊 Outputs
- EDA charts
- Model comparison plots
- SHAP explainability plots

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
```
